ForceCloseHandlesAsync(String, String, CancellationToken)

The ForceCloseHandlesAsync(String, String, CancellationToken) operation closes a handle or handles opened on a file at the service. It supports closing a single handle specified by handleId or or closing all handles opened on that resource.

This API is intended to be used alongside ListHandlesAsync(String, Nullable<Int32>, CancellationToken) to force close handles that block operations. These handles may have leaked or been lost track of by SMB clients. The API has client-side impact on the handle being closed, including user visible errors due to failed attempts to read or write files. This API is not intended for use as a replacement or alternative for SMB close.

For more information, see .

Optional MD5 hash of the range content. Must not be used when writeType is set to Clear.

This hash is used to verify the integrity of the range during transport. When this hash is specified, the storage service compares the hash of the content that has arrived with this value.
